# School Library experiment

## Wat heb ik gedaan?
In dit experiment heb ik gewerkt met de School Library REST API in de
DEVASC-omgeving. Met deze API kan ik boeken beheren in een digitale
schoolbibliotheek.

Ik heb geleerd hoe ik met verschillende API-endpoints kan werken om
boeken op te halen, toe te voegen en te verwijderen. Daarbij heb ik ook
beveiligde API-calls uitgevoerd met behulp van authenticatie en tokens.
